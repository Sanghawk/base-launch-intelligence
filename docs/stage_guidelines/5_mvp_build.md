# Stage 5 — MVP Build

**Goal:** Build the smallest usable version.

**What needs to be accomplished:**

- Build the ingestion pipeline for new Base token and pool events.
- Store normalized token, pool, deployer, and liquidity data.
- Add basic contract-risk enrichment.
- Add basic deployer-history tracking.
- Add a first-pass liquidity-quality score.
- Build a simple dashboard table showing launches, scores, risk flags, liquidity status, and reasons.
- Add alerting for high-priority launches or major risk/liquidity changes.
- Prioritize a working vertical slice over a polished product.

**Gate:** Can you use it yourself instead of opening DEX Screener first?
