# Base Launch Intelligence Console

Private Base-native launch intelligence and risk-ranking console MVP.

The product is intended to help an analyst/operator triage new Base token launches by quality, risk, liquidity quality, and deployer context.

## Status

Repository bootstrap only.

This repo currently contains root project metadata and placeholder directories. Workspace setup, app scaffolding, provider integrations, database schema, worker logic, and scoring logic are intentionally out of scope for this task.
